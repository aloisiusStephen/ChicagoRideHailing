Chicago Taxi Trips Fare Prediction¶

The objective of this work is to construct a model that can predict the fare for each trip. Given that the dataset for 2021 is not evenly distributed across the months; less data points in the earlier part of 2021 (likely due to the COVID situation earlier in the year), the dataset for 2020 is used as the training dataset. This also serves as a study on the possibility of forecasting future fares using past data.

Modelling Approach¶

The following models are presented in this report based on the accompanying justifications:

  1. Linear Regression. Close resemblance to the well-known taxi fare formula.
  2. LASSO Regression. An incremental improvement to the linear regression that includes regularization.
  3. XGBoost. A decision-tree ensemble method that often produce good results for small to medium sized problems.

Base Model¶

A quick search reveals that trip fares are traditionally calculated based on the following calculations:

$\text{Fare} = \text{Base Fare} + (\text{Trip Distance} \times \text{Fare Rate per Unit Distance}) + (\text{Trip Time} \times \text{Fare Rate per Unit Time}) $

The trip fare against distance and duration is plotted below to verify the applicability of this formulation in predicting the fares.

def evaluateModel(X,Y,model):
    prediction = model.predict(X)
    
    mse = mean_squared_error(Y,prediction)
    r2 = r2_score(Y,prediction)
    print('RMSE = {0:.2f} USD\n R2 = {1:.4f}'.format(mse**0.5,r2))
    return [mse,r2]

Feature Engineering¶

Datetime Features¶

The Datetime features are added by extracting the month, starting hour, and day of the week for each trip.

In [27]:
## add hour and day of week columns
def expandDateTimeFeatures(df_input):
    df_input['Trip Start Hour'] = df_input['Trip Start Timestamp'].dt.hour
    df_input['Trip Date'] = df_input['Trip Start Timestamp'].dt.date
    df_input['Trip Month'] = df_input['Trip Start Timestamp'].dt.month
    df_input['Trip Day of Week'] = df_input['Trip Start Timestamp'].dt.dayofweek
    
    return df_input

Latitiude and Longitude difference¶

Next the latitude and longitude of the pickup and dropoff points are engineered as a new feature to reflect the difference in latitude for pickup and dropoff, similarly for the longitude values, to reflect the direction of the trip.

def addLatLongFeature(df_input):
    df_input['Latitude Difference'] = df_input['Dropoff Centroid Latitude'] - df_input['Pickup Centroid Latitude']
    df_input['Longitude Difference'] = df_input['Dropoff Centroid Longitude'] - df_input['Pickup Centroid Longitude']

Conclusion¶

The prediction of taxi trip fares can be made relatively well using a linear regression model using just the trip distance and duration as features, with additional features giving it only marginal improvements. Regularization of the linear regression, in the form of the LASSO regression, produces a slightly worst performance. The XGBoost model can predict the fares extremely well, especially with the additional features.
